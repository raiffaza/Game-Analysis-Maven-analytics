# Game-Analysis-Maven-analytics
Video game sales analysis using Python and exploratory data analysis to uncover trends in genres, platforms, regional markets, and sales behavior from 1971 to 2020. This project demonstrates data cleaning, missing value handling, correlation analysis, and business driven insights using real world gaming data.
